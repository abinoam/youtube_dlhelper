⁠3. Aktualisieren

Falls Sie eine frühere Version von youtube_dlhelper benutzt haben, folgen Sie den folgenden Schritten:
  1. Gehen Sie zur Konsole/Terminal und trippen Sie:
  2. cd ~
  3. mkdir -p .config/youtube_dlhelper
  4. mv .youtube_dlhelper/youtube_dlhelper.conf .config/youtube_dlhelper/
  5. rm -rf .youtube_dlhelper
