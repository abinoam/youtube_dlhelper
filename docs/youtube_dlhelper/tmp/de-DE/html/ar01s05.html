⁠5. Benutzung

Nach der Installation tippen Sie youtube_dlhelper YoutubeURL.
⁠

Beispiel 1.

youtube_dlhelper http://www.youtube.com/watch?v=aHjpOzsQ9YI
Falls eine https:// URL nicht funktioniert, versuchen Sie http://.
Dannach fragt das Programm, ob es ein Interpret oder eine Gruppe ist. Falls es ein Interpret ist, wird der Vorname und Nachname abgefragt. Anschließend wird ein Verzeichnis in $MUSICDIR/surname_firstname/Youtube erstellt. Liegt ein Interpret vor, wird der Gruppenname als Verzeichnisname benutzt.
