⁠4. Die Konfigurationsdatei

Die Konfigurationsdatei liegt in /home/YOU/.config/youtube_dlhelper/youtube_dlhelper.conf. Sie sieht wie folgt aus:: 
musicdir = 'Musik'
ogg_file_accept = true
Sie vergeben den Namen Ihres Musikverzeichnisses. Tragen Sie "Musik" ein wenn ihr Verzeichnis in /home/sascha/Musik liegt. Falls ogg_file_accept zu true gesetzt ist, versucht das Programm, eine Konvertierung zu MP3.
